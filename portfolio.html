AgentSQL

💡 Assistente inteligente para análise de dados empresariais com IA.

Descrição:
O AgentSQL conecta uma LLM (IA) à base de dados da empresa via API. Com ele, gestores podem fazer perguntas em linguagem natural sobre vendas, faturamento, estoque ou métricas específicas. A IA interpreta a pergunta, consulta os dados automaticamente e retorna uma análise estratégica com comparações, resumos executivos e gráficos, de forma clara e visual.

Tecnologias: FastAPI · LangChain · PostgreSQL · OpenAI · Streamlit · Pandas

Webhook WhatsApp com IA

🤖 Assistente inteligente via WhatsApp

Descrição:
API desenvolvida em Python com FastAPI que conecta um número de WhatsApp Business a uma IA capaz de interpretar áudios e textos. A aplicação realiza buscas por ID do cliente, registra o histórico completo de conversas em banco de dados, e responde de forma personalizada utilizando a OpenAI.

Com integração ao sistema Webhook_Copiloto, ela permite atendimento humanizado, automatizado e com memória de contexto, ideal para suporte ao cliente, vendas e automações. O projeto também conta com controle de sessões, tokens, e persistência via banco de dados relacional.

Tecnologias: FastAPI · Python · PostgreSQL · OpenAI API · PM2 · Docker · WhatsApp Business API (Baileys) · Render
